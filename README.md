# BankingSystemSimulatorApplication
## Цель: 

создать  мини-приложение, имитирующее работу банковской системы, с использованием изученных тем (потоки, синхронизация, общие данные, исключения и т. д.).

### Функциональность:

1. Приложение должно иметь несколько потоков, которые будут имитировать работу клиентов, выполняющих банковские операции (внесение, снятие, перевод денег).
2. У каждого клиента-потока должен быть свой идентификатор и баланс на счету.
3. В приложении должен быть общий класс «Банк», в котором будет храниться информация обо всех клиентах и их балансах.
4. Каждый клиентский поток должен иметь возможность выполнять следующие операции: 
  * Перевод денег на свой счёт
  * Снятие денег со своего счета
  * Перевод денег на счёт другого клиента
5. Приложение должно использовать синхронизацию, чтобы избежать проблем с доступом к общим данным (балансам счетов).
6. Приложение должно обрабатывать исключения, которые могут возникнуть во время выполнения операций (например, на счёте недостаточно средств для выполнения операции).
7. В приложении должна быть функция «Статистика», которая будет выводить информацию обо всех клиентах и их балансах.
### Требования:
* Приложение должно быть написано на Java
* Приложение должно использовать потоки и синхронизацию для работы с общими данными
* Приложение должно обрабатывать исключения и ошибки
* Приложение должно иметь удобный для пользователя интерфейс (например, консольный или графический)
### Оценка:
* Правильность и работоспособность приложения (50 %)
* Использование изученных тем (30 %)
* Стиль и организация кода (20%)
### Дополнительные задания:
* Добавьте функцию «Аудит», которая будет записывать все операции, выполненные клиентами, в журнал.
* Добавьте функцию «Безопасность», которая будет проверять пароль клиента перед выполнением операции.
* Добавьте функцию «Уведомление», которая будет отправлять клиенту уведомления после выполнения операции (например, на электронную почту или телефон).
* Добавьте функцию «Отчётность», которая будет генерировать отчёты обо всех операциях, совершённых клиентами, за определённый период времени.
* Добавьте функцию «Управление клиентами», которая позволит администратору добавлять, удалять и редактировать клиентов.
* Добавьте функцию «Управление операциями», которая позволит администратору останавливать, возобновлять и отменять операции.
